# Portfolio - Next.js 14

A modern portfolio website built with Next.js 14, featuring animated components and a responsive design.

## Project Structure

```
portfolio/
├── app/
│   └── api/              # Backend API routes (reserved for future use)
├── components/            # React components
│   ├── About/
│   ├── AnimatedLetters/
│   ├── Contact/
│   ├── Home/
│   │   └── Logo/
│   ├── Layout/
│   └── Sidebar/
├── lib/                   # Database connection and utilities (reserved for future use)
├── pages/                 # Next.js pages
│   ├── _app.js           # App wrapper with global styles
│   ├── index.js          # Home page
│   ├── about.js          # About page
│   └── contact.js        # Contact page
├── public/                # Static assets
│   ├── fonts/            # Custom fonts
│   ├── images/           # Images and logos
│   └── favicon.ico
├── styles/                # Global styles
│   └── globals.css       # Tailwind CSS and custom styles
├── next.config.js        # Next.js configuration
├── tailwind.config.js    # Tailwind CSS configuration
├── postcss.config.js     # PostCSS configuration
└── package.json          # Dependencies and scripts
```

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server
- `npm run lint` - Run ESLint

## Features

- **Next.js 14** - Latest Next.js with App Router support
- **Tailwind CSS** - Utility-first CSS framework
- **Font Awesome** - Free icons
- **Animate.css** - Free CSS animations
- **React Leaflet** - Free map integration
- **EmailJS** - Free email service integration
- **React Loaders** - Free loading animations
